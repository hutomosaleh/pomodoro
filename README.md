# 🍷 Study logger bot

Discord Bot for tracking our study hours

## ⚙ Installation

### 1. Install required python libraries

```bash
pip install -r requirements.txt
```

### 2. Make .env file

Inside should contain
```bash
DISCORD_TOKEN=<your_bot_token>
MONGODB_TOKEN=<your_mongodb_token>
```

## 🧀 Usage

```bash
python main.py
```
